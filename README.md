# Inventory Management System

A full-stack Inventory Management System built using:

* React (Frontend)
* FastAPI (Backend)
* PostgreSQL (Database)
* Docker & Docker Compose

## Features

* Product Management

  * Create Product
  * Update Product
  * Delete Product
  * View Products

* Customer Management

  * Create Customer
  * Delete Customer
  * View Customers

* Order Management

  * Create Orders
  * View Order Details
  * Delete Orders

* Dashboard

  * Total Products
  * Total Customers
  * Total Orders
  * Low Stock Products

## Tech Stack

Frontend:

* React
* Axios
* Tailwind CSS

Backend:

* FastAPI
* SQLAlchemy

Database:

* PostgreSQL

Containerization:

* Docker
* Docker Compose

## Run Locally

```bash
docker compose up --build
```

Frontend:

```text
http://localhost:5173
```

Backend:

```text
http://localhost:8000/docs
```
